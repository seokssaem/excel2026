모듈 1 — 데이터 정리·정제

지저분한 원본 데이터를 분석 가능한 깨끗한 데이터로 만들기

💡왜 데이터 정제가 먼저인가?
❌ 정제 전 — 흔한 문제들
같은 민원이 2번 등록됨 (중복)
부서명이 "행정지원팀", "행정 지원팀", "행정지원" 혼재
날짜 형식이 "2025.3.7", "25/3/7" 뒤섞임
처리일수 칸에 "없음", "-", 공백 혼재
COUNTIFS 결과가 실제와 다름 → 보고서 오류
✅ 정제 후 — 달라지는 것들
피벗테이블 집계가 정확하게 동작
VLOOKUP·XLOOKUP 오류 없음
월별·부서별 통계가 신뢰 가능
차트가 실제 현황을 정확히 반영
보고서 재작업 0건 → 신뢰도 상승
🔁실습 1-1 — 중복 민원 접수번호 제거민원 데이터📂 민원현황_raw.xlsx
📌 실무 상황 민원팀에서 받은 엑셀 파일에 동일한 민원이 2번 등록되어 있습니다. 수기로 찾으면 30분 이상 걸리는 작업을 3초에 해결합니다.
📂 파일 열기: 민원현황_raw.xlsx → 민원현황 시트 | A열=접수번호, F열=처리상태, G열=처리일수

🖱️ 방법 A — 리본 메뉴 (초보자)

리본 메뉴 → 데이터 탭 클릭
3
기준 열 선택
"접수번호" 열만 체크 → 확인
→ "X개의 중복 값이 제거되었습니다" 메시지 확인

🔍 방법 B — 수식으로 중복 표시 후 확인

-- I2에 입력: 중복이면 "중복" 표시 (A열=접수번호 기준)
=IF(COUNTIF($A$2:A2,A2)>1,"중복","")

📌 실무 상황 D열(처리부서)에 "건설과", "건설 과", "건설팀" 등이 섞여 있어 피벗테이블 집계가 여러 개로 분리됩니다.

⚡ 방법 1 — 찾기·바꾸기 (Ctrl+H)

1
Ctrl+H 찾기·바꾸기 열기

3

⚡ 방법 2 — TRIM + SUBSTITUTE 수식

-- 앞뒤 공백 제거 (D열=처리부서)
=TRIM(D2)

-- "건설 과" → "건설과"로 치환
=SUBSTITUTE(D2,"건설 과","건설과")

-- 복합: 공백 제거 후 값만 붙여넣기
=TRIM(SUBSTITUTE(D2," ",""))

📅실습 1-3 — 날짜 형식 통일 + 경과일 자동 계산예산·민원 공통📂 민원현황_raw.xlsx
📂 파일 열기: 민원현황_raw.xlsx → B열(접수일자) 확인 — "2025-03-07" / "2025.03.07" / "25/03/07" 세 가지가 혼재되어 있습니다

📌 STEP 1 — 왜 날짜 통일이 필요한가?

엑셀이 날짜로 인식하지 못하면 셀 왼쪽 정렬(텍스트 취급)로 표시됩니다. 이 상태에서는 TODAY()-B2 같은 날짜 계산 수식이 #VALUE! 오류를 냅니다. 빠른 확인법: B열 날짜 셀 클릭 → 수식 입력줄에 숫자(예: 45726)가 보이면 ✅ 날짜형, 텍스트 그대로 보이면 ❌ 텍스트형

🔧 STEP 2 — 날짜 형식 유형별 통일 방법

▶ 케이스 A — "2025.03.07" 점 구분 형식

1
B열 선택 → Ctrl+H
찾을 내용: . → 바꿀 내용: - → 모두 바꾸기
2
셀 서식 → 날짜 형식 확인
Ctrl+1 → 표시 형식 → 날짜 선택

▶ 케이스 B — "25/03/07" 2자리 연도 형식

-- "25/03/07" → 진짜 날짜로 변환 (빈 열에 입력)
=DATE(2000+LEFT(B2,2)*1,
     MID(B2,4,2)*1,
     RIGHT(B2,2)*1)

▶ 케이스 C — DATEVALUE 만능 변환

-- 텍스트 날짜를 진짜 날짜로 한방에 변환
-- "2025-03-07", "2025/03/07" 모두 처리 가능
=DATEVALUE(B2)

-- 변환 후 → 값만 붙여넣기로 원본 교체
-- C열에 수식 입력 → C열 복사 → B열에 Ctrl+Shift+V → 값만
⚠️ DATEVALUE 결과는 숫자(일련번호)입니다. 붙여넣기 후 반드시 셀 서식을 날짜로 바꿔주세요!
Ctrl+1 → 표시 형식 → 날짜

▶ 텍스트→날짜 마법사 (가장 쉬운 방법)

1
B열 선택 → [데이터] → [텍스트 나누기]
2
3단계에서 "날짜" 선택 → YMD 설정 → 마침
엑셀이 자동으로 날짜 형식으로 변환!

📐 STEP 3 — 날짜 통일 후 경과일·기한 자동 계산

-- 접수일(B열)로부터 오늘까지 경과일
=TODAY()-B2

-- 처리기한까지 남은 일수 (14일 기준)
-- 음수면 기한 초과!
=B2+14-TODAY()

-- 공문서 날짜 형식으로 표시
=TEXT(B2,"YYYY년 MM월 DD일")

-- 이번 달 접수 건수 (B열 기준)
=COUNTIFS(B:B,">="&EOMONTH(TODAY(),-1)+1,
          B:B,"<="&EOMONTH(TODAY(),0))

🔍함수 2-1 — XLOOKUP : 부서코드 → 부서명·팀장 자동 매핑인사·예산 공통📂 인사근태.xlsx + 부서코드표.xlsx
📌 실무 상황 인사근태.xlsx의 C열(부서코드)에는 A001, B003 같은 코드만 있습니다. 부서코드표.xlsx의 B열(부서명)·C열(팀장명)을 참조해 수식으로 부서명을 자동 채웁니다. 300명 기준 2시간 → 3초.
📂 파일 구조: 인사근태.xlsx C열=부서코드 | 부서코드표.xlsx A열=코드, B열=부서명, C열=팀장명, D열=내선번호
-- 인사근태 D열에 입력: C열 부서코드로 부서명 찾기
=XLOOKUP(C2, 부서코드표!A:A, 부서코드표!B:B, "미등록")

-- 심화: 부서명·팀장명·내선번호 3개 열 동시 반환 (B:D)
=XLOOKUP(C2, 부서코드표!A:A, 부서코드표!B:D, "미등록")
   -- B:D = 부서명(B)·팀장명(C)·내선번호(D) 동시 반환

-- 역방향: 부서명으로 코드 역추적
=XLOOKUP(D2, 부서코드표!B:B, 부서코드표!A:A, "코드없음")
📋 XLOOKUP vs VLOOKUP 핵심 차이
VLOOKUP은 왼쪽→오른쪽만 가능, 열 번호 수동 입력 필요
XLOOKUP은 양방향, 오류 처리 내장, 열 추가해도 수식 안 깨짐
엑셀 2019 이상에서 사용 가능 (2016은 INDEX+MATCH)

엑셀 2016 사용자용 — INDEX+MATCH 대체식

=INDEX(부서코드표!B:B,
  MATCH(C2, 부서코드표!A:A, 0))
💰함수 2-2 — SUMIFS·COUNTIFS : 다중 조건 집계 자동화예산·민원 공통📂 예산집행현황.xlsx · 민원현황_raw.xlsx
📌 실무 상황 "3분기 건설과 예산집행액 합계", "이번 달 도로 관련 완료 민원 건수"를 필터 걸고 수기 계산하면 각 조합마다 5분. 20가지 조합 = 100분 → 수식 하나로 해결.
📂 예산집행현황.xlsx: A=부서명, C=예산액, D=집행액, E=분기, F=집행률(%), G=미집행액
📂 민원현황_raw.xlsx: B=접수일자, C=민원유형, D=처리부서, F=처리상태
-- SUMIFS: A=부서명, D=집행액, E=분기 (예산집행현황.xlsx)
=SUMIFS(D:D, A:A, "건설과", E:E, "3분기")

-- SUMIFS: F=집행률(%), G=미집행액 — 집행률 70% 미만 미집행액 합산
=SUMIFS(G:G, F:F, "<70")

-- COUNTIFS: B=접수일자, C=민원유형, F=처리상태 (민원현황_raw.xlsx)
=COUNTIFS(B:B,">="&DATE(2025,3,1),
           B:B,"<"&DATE(2025,4,1),
           C:C,"도로/교통",
           F:F,"완료")

-- 집행률 자동 계산 (A=부서명, C=예산액, D=집행액)
=SUMIFS(D:D,A:A,"건설과")
  /SUMIFS(C:C,A:A,"건설과")*100
📤 활용 예시 — 예산집행현황.xlsx 기준 건설과 3분기 집행액: =SUMIFS(D:D,A:A,"건설과",E:E,"3분기") → 예: 423,000천원
복지과 집행률: =SUMIFS(D:D,A:A,"복지과")/SUMIFS(C:C,A:A,"복지과")*100 → 예: 67.0%
3월 완료 민원: =COUNTIFS(B:B,">="&DATE(2025,3,1),F:F,"완료") → 예: 147건
⚡함수 2-3 — 동적 배열 수식 : UNIQUE·SORT·FILTER엑셀 365/2021📂 민원현황_raw.xlsx
📌 실무 상황 F열(처리상태)="처리중"인 민원 목록만 자동 추출해 별도 시트에 표시합니다. 데이터가 바뀌면 자동으로 갱신됩니다.
📂 파일 열기: 민원현황_raw.xlsx | C열=민원유형, F열=처리상태, G열=처리일수
-- FILTER: F열(처리상태)="처리중"인 민원만 자동 추출
=FILTER(A2:H500, F2:F500="처리중", "해당없음")

-- UNIQUE: C열(민원유형) 목록 자동 추출 (중복 제거)
=UNIQUE(C2:C500)

-- SORT: G열(처리일수) 많은 순으로 자동 정렬
=SORT(A2:H500, 7, -1)
   -- 7번째 열(G=처리일수) 기준, -1은 내림차순

-- 복합: 처리중 + 처리일수 7일 초과만 추출 후 정렬
=SORT(FILTER(A2:H500,
    (F2:F500="처리중")*(G2:G500>7)),7,-1)
⚠️ FILTER·UNIQUE·SORT는 엑셀 365 / 2021 이상에서만 지원됩니다. 2019 이하라면 피벗테이블이나 고급 필터로 대체하거나, AI에게 "엑셀 2019 버전으로 같은 결과를 내는 수식 알려줘"라고 요청하세요.
🚦함수 2-4 — IF·IFS·IFERROR : 오류 없는 스마트 판단 수식모든 버전📂 예산집행현황.xlsx · 인사근태.xlsx
📂 예산집행현황.xlsx: F열=집행률(%) | 민원현황_raw.xlsx: F열=처리상태, G열=처리일수
-- 예산집행현황_blank.xlsx H열에 입력: F열(집행률) 기준 등급 판정
=IFS(F2>=90%,"🟢 우수",
     F2>=70%,"🟡 보통",
     F2>=0%,"🔴 부진")

-- 인사근태_blank.xlsx C열에 입력 / IFERROR: 부서코드 없을 때 대체값 표시
=IFERROR(XLOOKUP(C2,부서코드표!A:A,부서코드표!B:B),"코드확인요망")

-- 민원현황_raw.xlsx — AND 조건: F열=처리상태, G열=처리일수
=IF(AND(F2="처리중",G2>7),"⚠️ 즉시처리","")

📊 예산·집행 분석

✨ 모든 AI
집행률 자동 계산 + 등급 판정
엑셀 수식 만들어줘. 시트 구성: A열=부서명, B열=사업코드, C열=예산액(천원), D열=집행액(천원), E열=분기(1분기/2분기...) 원하는 수식: 1. F열: 집행률(%) = 집행액/예산액×100, 소수점 1자리 → =D2/C2*100 2. H열: 집행률(F열) 90% 이상이면 "🟢우수", 70~89%면 "🟡보통", 70% 미만이면 "🔴부진" 3. G열: 미집행액 = 예산액 - 집행액 → =C2-D2 4. 전체 집행률 요약: SUMIFS로 건설과만 집행액 합산하는 수식 조건: 엑셀 2019에서 동작, 각 수식 옆에 한 줄 설명 포함
✨ 모든 AI
부서별 예산 현황판 자동 생성
엑셀로 예산 현황판 만드는 수식 알려줘. 원본 데이터: 예산집행현황.xlsx 열 구성: A=부서명, B=사업코드, C=예산액(천원), D=집행액(천원), E=분기, F=집행률(%), G=미집행액(천원) 현황판에 필요한 수식: - B2: 건설과 전체 예산 합계 → =SUMIFS(C:C,A:A,"건설과") - C2: 건설과 전체 집행액 합계 → =SUMIFS(D:D,A:A,"건설과") - D2: 건설과 집행률(%) - 각 부서별로 같은 구조 (건설과/복지과/환경과/교육지원과) 엑셀 2019 버전, SUMIFS 사용, 수식만 나열해줘

집행 부진 사업 자동 추출
엑셀에서 조건에 맞는 행을 자동으로 추출하는 방법 알려줘. 원본: A열=부서명, B열=사업명, C열=집행률(%), D열=분기 원하는 결과: 별도 시트에 집행률 70% 미만인 행만 자동으로 표시 엑셀 버전별로 방법 알려줘: 1. 엑셀 365: FILTER 함수 사용 2. 엑셀 2019: 다른 방법으로 각 방법의 수식과 사용법 설명해줘

✨ 모든 AI
민원 유형별·월별 자동 집계
엑셀 COUNTIFS 수식 만들어줘. 파일: 민원현황_raw.xlsx 열 구성: A=접수번호, B=접수일자(날짜형식), C=민원유형, D=처리부서, F=처리상태, G=처리일수 원하는 수식들: 1. 3월 도로/교통 완료 건수 → C열="도로/교통", F열="완료", B열=3월 2. 처리일수(G열) 7일 초과 + 처리중(F열) 건수 3. 이번 달 전체 접수 건수 (B열, TODAY 함수 활용) 4. 처리부서(D열)별 평균 처리일수 (완료 건만) 엑셀 2019 버전, 수식과 설명 같이 써줘

VBA
Sub 월간보고서자동화()
  Dim ws As Worksheet
  Dim fileName As String
  Dim savePath As String

  ' 현재 활성 시트 대상
  Set ws = ActiveSheet

  ' 화면 업데이트 끄기 (속도 향상)
  Application.ScreenUpdating = False

  ' 1. 헤더 행(1행) 서식
  With ws.Rows(1)
    .Interior.Color = RGB(13, 31, 60)   ' 네이비 배경
    .Font.Color = RGB(255, 255, 255)    ' 흰 글자
    .Font.Bold = True
    .Font.Size = 11
  End With

  ' 2. 데이터 행 줄무늬 (짝수 행 연한 파란색)
  Dim lastRow As Long
  lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row
  Dim i As Long
  For i = 2 To lastRow
    If i Mod 2 = 0 Then
      ws.Rows(i).Interior.Color = RGB(239, 246, 255)
    Else
      ws.Rows(i).Interior.Color = RGB(255, 255, 255)
    End If
  Next i

  ' 3. 전체 열 너비 자동 맞춤
  ws.Cells.EntireColumn.AutoFit

  ' 4. 오늘 날짜로 파일명 저장
  fileName = "월간보고서_" & Format(Date, "YYYY년MM월") & ".xlsx"
  savePath = ThisWorkbook.Path & "\" & fileName
  ThisWorkbook.SaveAs savePath, xlOpenXMLWorkbook

  Application.ScreenUpdating = True
  MsgBox "✅ 완료! 저장위치: " & savePath
End Sub

② 처리기한 초과 민원 자동 강조 + 목록 추출 📂 민원현황_raw.xlsx

VBA
Sub 기한초과민원추출()
  Dim ws As Worksheet
  Dim wsResult As Worksheet
  Set ws = Sheets("민원현황")

  ' 결과 시트 새로 만들기 (있으면 삭제 후 재생성)
  Application.DisplayAlerts = False
  On Error Resume Next
  Sheets("기한초과목록").Delete
  On Error GoTo 0
  Application.DisplayAlerts = True

  Sheets.Add(After:=Sheets(Sheets.Count)).Name = "기한초과목록"
  Set wsResult = Sheets("기한초과목록")

  ' 헤더 복사
  ws.Rows(1).Copy wsResult.Rows(1)

  Dim lastRow As Long
  lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row
  Dim resultRow As Long
  resultRow = 2
  Dim count As Long
  count = 0

  Dim i As Long
  For i = 2 To lastRow
    ' F열(6)=처리상태, G열(7)=처리일수 (민원현황_raw.xlsx 열 구조)
    Dim 처리상태 As String
    Dim 처리일수 As Variant
    처리상태 = ws.Cells(i, 6).Value
    처리일수 = ws.Cells(i, 7).Value

    ' 처리중이거나 처리일수 7일 초과인 행
    If 처리상태 = "처리중" Or (IsNumeric(처리일수) And 처리일수 > 7) Then
      ws.Rows(i).Copy wsResult.Rows(resultRow)
      ' 빨간색 강조
      wsResult.Rows(resultRow).Interior.Color = RGB(254, 242, 242)
      resultRow = resultRow + 1
      count = count + 1
    End If
  Next i

  wsResult.Cells.EntireColumn.AutoFit
  MsgBox "⚠️ 기한초과 민원: " & count & "건 추출 완료"
End Sub
