필수 실무 함수
VLOOKUP
표에서 값 찾아오기
=VLOOKUP(찾는값,범위,열번호,0)
XLOOKUP
왼쪽 참조도 가능 (M365)
=XLOOKUP(값,검색범위,반환범위)
SUMIFS
다중 조건 합계
=SUMIFS(합계범위,조건1범위,조건1,...)
COUNTIFS
다중 조건 카운트
=COUNTIFS(범위1,조건1,범위2,조건2)
IFERROR
오류 발생 시 대체값
=IFERROR(수식, "없음")
DATEDIF
날짜 차이 계산
=DATEDIF(시작일,종료일,"Y")
SUBTOTAL
필터 상태에서 합계
=SUBTOTAL(9,범위) ← 9=합계
TEXT
숫자를 원하는 형식으로
=TEXT(날짜,"YYYY-MM-DD")
주요 오류 & 해결법
#N/A
찾는 값이 없음
→ IFERROR 감싸기, 데이터 공백 확인
#VALUE!
데이터 형식 불일치
→ 숫자로 저장된 텍스트 확인 (TRIM)
#REF!
참조 범위가 삭제됨
→ 수식 재확인, 행/열 삭제 전 확인
#DIV/0!
0으로 나누기
→ =IFERROR(A/B, 0) 또는 IF(B=0,"",A/B)
#NAME?
함수명 오타
→ 함수명 확인, 큰따옴표 누락 확인
